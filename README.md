Given a string made up of letters a, b, and/or c, switch the position of letters a and b (change a to b and vice versa). Leave any incidence of c untouched.

Example:

'acb' --> 'bca'
'aabacbaa' --> 'bbabcabb'

INPUT       |     OUTPUT  
abc         |       bac
aaabcc      |       baaacc
a           |       a
b           |       b
c           |       c
ab          |       ba
ac          |       ac
bc          |       bc
